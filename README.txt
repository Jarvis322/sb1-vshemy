Dijital Takipçi - Abonelik Yönetim Platformu
==========================================

Dijital Takipçi, dijital aboneliklerinizi akıllıca yönetmenizi sağlayan modern bir web uygulamasıdır. Streaming servisleri, müzik platformları, oyun abonelikleri ve diğer dijital hizmetleri tek bir yerden takip edebilir, maliyetlerinizi analiz edebilir ve kişiselleştirilmiş içerik önerileri alabilirsiniz.

Özellikler
----------

1. Abonelik Yönetimi
   - Tüm dijital aboneliklerinizi tek platformda yönetin
   - Detaylı maliyet takibi ve analizi
   - Kategori bazlı filtreleme
   - Abonelik ekleme ve silme
   - Yenileme tarihi hatırlatmaları

2. İçerik Önerileri
   - Mood Testi ile ruh halinize uygun içerik önerileri
   - Şans Topu ile rastgele içerik keşfi
   - Netflix, Amazon Prime ve BluTV içeriklerini kapsayan öneriler
   - Kişiselleştirilmiş tavsiyeler

3. Analitik Özellikler
   - Görselleştirilmiş maliyet grafikleri
   - Kategori bazlı harcama analizi
   - Aylık ve yıllık maliyet karşılaştırmaları
   - Tasarruf önerileri

4. Kullanıcı Deneyimi
   - Modern ve sezgisel arayüz
   - Karanlık/Aydınlık tema desteği
   - Mobil uyumlu tasarım
   - Animasyonlu geçişler ve etkileşimler

Teknolojiler
------------
- React
- TypeScript
- Tailwind CSS
- Chart.js
- Framer Motion
- Lucide Icons
- React Hot Toast
- Zustand
- Supabase

Kurulum
-------
1. Projeyi klonlayın
2. npm install ile bağımlılıkları yükleyin
3. npm run dev ile geliştirme sunucusunu başlatın

Kullanım
--------
1. Hesap oluşturun veya giriş yapın
2. Aboneliklerinizi ekleyin
3. Dashboard üzerinden analizleri inceleyin
4. Mood Testi ile içerik önerileri alın
5. Şans Topu ile yeni içerikler keşfedin
